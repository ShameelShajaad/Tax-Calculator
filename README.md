## üßæ Description
This Java program is a **menu-driven salary, tax, and leasing management system**.  
It allows employees and users to calculate various types of taxes, social contributions, and leasing payments.  

The system includes:

- **Withholding Tax** (Rent, Bank Interest, Dividend)  
- **Payable Tax**  
- **Income Tax**  
- **Social Security Contribution Levy (SSCL) Tax**  
- **Leasing Payment** (Monthly Installments, Leasing Category, and Find Lease Amount)  

It also provides **input validation** and a **clear menu structure** for easy navigation.
